# 01 -- DEMAND CREATION (TRAFFIC) -- Complete Breakdown

## 1. Core Actions

### A. Intent-Aligned Traffic Sources

**What must happen:**
- Traffic sourced from high-intent channels (Search, Maps, Referrals, Directories)

**Why this is required:**
PPO dentistry is intent-driven. Low-intent traffic creates noise.

**What breaks if missing:**
- Junk calls
- Intake friction
- Marketing blamed incorrectly

---

## 2. Ownership (Human vs System)

**Human-owned:**
- Channel strategy
- Budget decisions

**System-owned:**
- Routing logic
- Entry point enforcement

**Why this split is required:**
Marketing creates demand. Operations convert it.

---

## 3. Failure Modes

- Low-intent channels
- Too many CTAs
- Wrong-location routing
- Traffic judged without downstream context

---

## 4. Recovery Logic

- Low conversion → Audit Speed-to-Lead
- Wrong routing → Enforce location logic
- 'Bad leads' complaints → Trace downstream handling

---

## 5. Outputs

Demand Creation must output:
- Qualified inbound demand
- Accurate location attribution
- Clean handoff to Speed-to-Lead
