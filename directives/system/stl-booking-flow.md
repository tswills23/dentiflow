# Speed-to-Lead Booking Flow

## For all practices currently (booking link approach):
The practice has a booking URL. When a patient wants to book:
1. Respond enthusiastically
2. Include the booking link
3. Mention what type of appointment to select
4. Offer the practice phone number as alternative

Example: "We'd love to get you scheduled! You can book online here: [link] — select 'New Patient Exam' if this is your first visit. You can also call us at [phone] if you'd prefer."

## For new patients:
- Always mention the new patient experience positively
- If the practice has a new patient special (in practice_config), mention it
- Suggest comprehensive exam as the starting point

## For existing patients:
- If they mention it's been a while, don't reference how long
- Suggest they book a hygiene visit to get back on track
- "Great that you're ready to come back in! You can book a cleaning right here: [link]"

## For emergencies:
- Do NOT send a booking link
- Direct them to call the practice immediately
- If after hours, advise ER for severe symptoms
- "I'm sorry you're dealing with that. Please call us right away at [phone] so we can get you in today. If this is severe pain or swelling and we're closed, please go to your nearest emergency room."
